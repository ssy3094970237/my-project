# 猫咪购物园 (Cat Shopping Park) - 微信小程序

这是一个专为猫咪爱好者和宠物主人打造的猫咪用品展示、导购与购物模拟的微信小程序。项目提供了生动的产品分类展示、购物车管理、收货地址管理、订单模拟以及线下参观预约等功能。

---

## 🌟 项目亮点

1. **丰富的商品分类**：包括猫粮、罐头、零食和玩具四大类，每一类都有精心准备的商品卡片。
2. **完整的交易闭环模拟**：用户可点击商品进入详情页查看服务保障，加入购物车，调整购买数量，并选择/添加收货地址来提交并结算订单。
3. **精美的 UI/UX 体验**：使用自然清爽的绿灰色调（`#93a07d`），提供丝滑的 Tab 页切换及交互 micro-animations。
4. **线下预约服务**：支持预约线下参观购物园体验区。

---

## 🛠️ 技术栈

- **前端技术**：微信小程序原生框架（WXML / WXSS / JavaScript / JSON）
- **页面路由**：多达 8 个核心路由目录，完全满足“至少3个独立路由”的课程要求
- **接口方案**：纯前端逻辑实现，配合 Mock API 接口文档进行模拟交互

## 🚀 安装与运行指南

### 1. 准备工作
- 确保已下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
- 本地克隆或解压本项目包。

### 2. 导入项目
1. 打开 **微信开发者工具**。
2. 点击 **导入** 按钮。
3. **项目路径** 选择解压后的根目录：`猫咪购物园`。
4. **AppID** 选择“测试号”或填写您个人的微信小程序 AppID。
5. 开发模式选择 **小程序**，点击 **确定** 导入。

### 3. 本地运行与预览
- 导入成功后，微信开发者工具将自动编译并展示预览界面。
- 可在模拟器中体验：商品选购 -> 加入购物车 -> 提交订单 -> 地址管理等功能。
- 点击顶部“真机调试”或“预览”，扫描生成的二维码即可在手机端进行实体机体验。
