# 猫咪购物园 - Mock API 接口文档

本项目为纯前端微信小程序项目，为满足工程规范与考核要求，本文档提供了模拟（Mock）的后端 API 接口设计方案。接口均遵循 RESTful API 设计规范，返回格式统一为 JSON。

---

## 统一响应格式说明

所有接口的响应均包含以下通用字段：

| 字段名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `code` | Integer | 状态码（`200` 表示成功，其它表示失败） |
| `message` | String | 提示信息 |
| `data` | Object/Array | 业务数据主体 |

---

## 1. 获取商品列表接口

### 接口基本信息
- **请求路径**：`/api/goods`
- **请求方法**：`GET`
- **接口描述**：获取猫咪购物园内的商品列表数据（支持分类查询）。

### 请求参数（Query）
| 参数名 | 类型 | 是否必填 | 默认值 | 示例值 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `category` | String | 否 | `all` | `food` | 分类：`food`（猫粮）、`can`（罐头）、`snacks`（零食）、`play`（玩具） |

```json
      "shop": "小猫爱吃鱼",
      "bao": "极速退款 退货运费险 七天无理由退货"
    },
    {
      "id": 2,
      "name": "金枪鱼胡萝卜罐",
      "price": "12.00",
      "num": 1,
      "desc": "本款产品包含新鲜金枪鱼与有机胡萝卜",
      "image": "/images/shopping-罐头3.png",

      "bao": "极速退款 退货运费险 七天无理由退货"

```

### 异常场景说明（500 - 服务器异常）
- **触发场景**：数据库连接失败或网络延迟。
- **响应示例**：
```json
{
  "code": 500,
  "message": "获取商品列表失败，系统繁忙，请稍后再试",
  "data": null
}
```

## 2. 提交订单接口

### 接口基本信息
- **请求路径**：`/api/orders`
- **请求方法**：`POST`
- **接口描述**：提交用户选购的商品并生成订单。

### 请求参数（JSON Body）
| 参数名 | 类型 | 是否必填 | 示例值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| `goods_list` | Array | 是 | `[{"id": 1, "num": 2}]` | 购买商品列表 |
| `address_id` | Integer | 是 | `102` | 收货地址ID |
| `total_price`| String | 是 | `"20.00"` | 订单总金额 |

### 响应示例（成功 - 200）
```json
{
  "code": 200,
  "message": "订单提交成功",
  "data": {
    "order_id": "ORD20260714174246",
    "status": "pending_payment",
    "create_time": "2026-07-14 17:42:46",
    "total_price": "20.00"
  }
}
```

### 异常场景说明（400 - 库存不足）
- **触发场景**：选购的某款商品库存已被抢光。
- **响应示例**：
```json
{
  "code": 400,
  "message": "提交订单失败：商品[羊奶鸡肉罐]库存不足",
  "data": null
}
```

## 3. 保存收货地址接口

### 接口基本信息
- **请求路径**：`/api/address`
- **请求方法**：`POST`
- **接口描述**：添加或修改用户的收货地址。

### 请求参数（JSON Body）
| 参数名 | 类型 | 是否必填 | 示例值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| `id` | Integer | 否 | `102` | 地址ID（传入则为修改，不传则为新增） |
| `name` | String | 是 | `张三` | 收货人姓名 |
| `phone` | String | 是 | `13800138000` | 联系电话 |
| `address` | String | 是 | `北京市朝阳区猫咪幸福里小区3号楼` | 详细地址 |

### 响应示例（成功 - 200）
```json
{
  "code": 200,
  "message": "地址保存成功",
  "data": {
    "id": 102,
    "name": "张三",
    "phone": "13800138000",
    "address": "北京市朝阳区猫咪幸福里小区3号楼"
  }
}
```

### 异常场景说明（422 - 参数校验失败）
- **触发场景**：手机号格式不正确或地址为空。
- **响应示例**：
```json
{
  "code": 422,
  "message": "地址保存失败：手机号格式不正确",
  "data": null
}
```

## 4. 预约参观接口

### 接口基本信息
- **请求路径**：`/api/visit`
- **请求方法**：`POST`
- **接口描述**：用户预约线下参观猫咪购物园/线下体验中心。

### 请求参数（JSON Body）
| 参数名 | 类型 | 是否必填 | 示例值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| `name` | String | 是 | `李四` | 预约人姓名 |
| `phone` | String | 是 | `13900139000` | 预约电话 |
| `visit_date`| String | 是 | `2026-07-20` | 计划参观日期 |
| `visitor_num`| Integer| 是 | `2` | 参观人数 |

### 响应示例（成功 - 200）
```json
{
  "code": 200,
  "message": "预约成功，我们将于参观前一天与您电话确认",
  "data": {
    "appointment_id": "APT8849201",
    "status": "confirmed"
  }
}
```

### 异常场景说明（400 - 预约人数超限）
- **触发场景**：所选日期预约名额已满。
- **响应示例**：
```json
{
  "code": 400,
  "message": "预约失败：该日期预约已满，请选择其他时间",
  "data": null
}
```
